<Header>Use localhost</Header>

In this tutorial, we'll create a test with `localhost`.

This is helpful for:

- testing a new feature while you are developing it
- adding test attributes like `data-qa=search-button` to your site while creating a test

<SubHeader>Local Runner</SubHeader>

First we need to start a local runner.

Make sure you have <a href="https://www.docker.com/products/docker-desktop" target="_blank">Docker</a> and <a href="https://nodejs.org/en/download/" target="_blank">Node.js</a> installed on your computer.

Now run `npx qawolf@latest` to start the runner.

<Image
  alt="Start the runner"
  height={247}
  src="/docs/use-localhost/start-runner.png"
  width={800}
/>

To stop the runner, press `CTRL+C`.

<SubHeader>Test localhost</SubHeader>

To connect to the runner add `?local=1` to the test URL.

For example change `https://www.qawolf.com/test/ckk4tp4qp00053w3q0j5xd9cd` to `http://qawolf.com/test/ckk4tp4qp00053w3q0j5xd9cd?local=1`.

When you enter `localhost` into the URL prompt, we automatically add `?local=1` for you.

<Image
  alt="Enter the localhost URL"
  height={204}
  src="/docs/use-localhost/enter-url.png"
  width={400}
/>
